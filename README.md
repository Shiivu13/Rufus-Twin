## 🛠️ How It Works (The Magic Behind the Scenes)

For the tech-curious, here is a simple breakdown of how the magic happens:

1. **The Brain (Gemini AI):** We use Google's advanced AI to understand your questions and write helpful, human-like responses.
2. **The Eyes (Tavily Search):** When you ask about a specific product, Rufus instantly searches the internet, reads articles and reviews, and summarizes the best information.
3. **The Face (React & Vite):** The beautiful, sleek, glass-like design you see on your screen.
4. **The Engine (FastAPI & Supabase):** The powerful backend system that connects your questions to the AI and stores the data securely.

## 🚀 Running It Locally (For Developers)

If you're a developer and want to run this project on your own computer, here is a quick setup guide:

### 1. Setup the Backend
Open your terminal and run these commands:
```bash
cd backend
pip install -r requirements.txt
# Create a .env file and add your API keys (Gemini, Supabase, Tavily)
uvicorn main:app --reload
```

### 2. Setup the Frontend
Open a new terminal window:
```bash
cd frontend
npm install
npm run dev
```
Now, open `http://localhost:5173` in your browser and you're good to go!
